# 🛡️ Network Security Quiz

An interactive, high-tech web-based quiz application designed to test and improve knowledge in Network Security. Built with a stunning futuristic "Cyberpunk" aesthetic, featuring smooth animations and a responsive interface.

## ✨ Features
- **Cyberpunk UI/UX:** A visually immersive experience with neon glows, glassmorphism, and smooth transitions.
- **Interactive Quiz:** Real-time feedback on answers.
- **Responsive Design:** Optimized for Desktops, Tablets, and Mobile devices.
- **Dynamic Questioning:** Engaging network security challenges.
- **Performance Optimized:** Fast loading times and lightweight codebase.

---

## 🛠️ Technologies Used
- **Frontend:** HTML5, CSS3 (Vanilla CSS), JavaScript (Vanilla JS).
- **Typography:** Google Fonts (Rajdhani, Exo 2, Share Tech Mono).
- **Deployment:** Firebase Hosting.
